# Quickstart

Get Torph up and running in your project in just a few steps. This guide will have you animating text in under 2 minutes.

<Steps>
  <Step title="Install Torph">
    Install the package using your preferred package manager:

    <CodeGroup>
    ```bash npm
    npm install torph
    ```

    ```bash yarn
    yarn add torph
    ```

    ```bash pnpm
    pnpm add torph
    ```
    </CodeGroup>
  </Step>

  <Step title="Import and use">
    Choose your framework and start animating:

    <CodeGroup>
    ```tsx React
    import { TextMorph } from "torph/react";
    import { useState, useEffect } from "react";

    const texts = [
      "Animate Text Easily",
      "Animate Text Effortlessly",
      "Animate Text Seamlessly",
      "Fluidly Animate Text",
    ];

    export default function App() {
      const [index, setIndex] = useState(0);

      useEffect(() => {
        const interval = setInterval(() => {
          setIndex((i) => (i + 1) % texts.length);
        }, 2000);
        return () => clearInterval(interval);
      }, []);

      return (
        <div style={{ fontSize: "2rem" }}>
          <TextMorph>{texts[index]}</TextMorph>
        </div>
      );
    }
    ```

    ```vue Vue
    <template>
      <div style="font-size: 2rem">
        <TextMorph :text="text" />
      </div>
    </template>

    <script setup lang="ts">
    import { ref, onMounted, onUnmounted, computed } from "vue";
    import { TextMorph } from "torph/vue";

    const texts = [
      "Animate Text Easily",
      "Animate Text Effortlessly",
      "Animate Text Seamlessly",
      "Fluidly Animate Text",
    ];

    const index = ref(0);
    let interval: ReturnType<typeof setInterval>;

    onMounted(() => {
      interval = setInterval(() => {
        index.value = (index.value + 1) % texts.length;
      }, 2000);
    });

    onUnmounted(() => {
      clearInterval(interval);
    });

    const text = computed(() => texts[index.value]);
    </script>
    ```

    ```svelte Svelte
    <script lang="ts">
      import { onMount, onDestroy } from 'svelte';
      import { TextMorph } from 'torph/svelte';

      const texts = [
        "Animate Text Easily",
        "Animate Text Effortlessly",
        "Animate Text Seamlessly",
        "Fluidly Animate Text",
      ];

      let index = 0;
      let interval: ReturnType<typeof setInterval>;

      onMount(() => {
        interval = setInterval(() => {
          index = (index + 1) % texts.length;
        }, 2000);
      });

      onDestroy(() => {
        clearInterval(interval);
      });

      $: text = texts[index];
    </script>

    <div style="font-size: 2rem">
      <TextMorph {text} />
    </div>
    ```

    ```typescript Vanilla JS
    import { TextMorph } from "torph";

    const texts = [
      "Animate Text Easily",
      "Animate Text Effortlessly",
      "Animate Text Seamlessly",
      "Fluidly Animate Text",
    ];

    const container = document.getElementById("torph-text")!;
    container.style.fontSize = "2rem";

    const morph = new TextMorph({ element: container });

    let index = 0;
    morph.update(texts[index]);

    setInterval(() => {
      index = (index + 1) % texts.length;
      morph.update(texts[index]);
    }, 2000);
    ```

    ```html HTML
    <!DOCTYPE html>
    <html>
      <head>
        <title>Torph Example</title>
      </head>
      <body>
        <div id="torph-text"></div>
        <script type="module" src="./main.ts"></script>
      </body>
    </html>
    ```
    </CodeGroup>
  </Step>

  <Step title="Enjoy the magic">
    That's it! Your text will now smoothly morph between values with beautiful animations.

    <Tip>
      The animation automatically respects the user's `prefers-reduced-motion` setting for better accessibility.
    </Tip>
  </Step>
</Steps>

## What's happening?

When the text changes, Torph:

1. **Segments the text** into characters or words using `Intl.Segmenter` (with fallback)
2. **Tracks each segment** with a unique ID to maintain identity across changes
3. **Animates positions** using FLIP technique (First, Last, Invert, Play)
4. **Fades in/out** new and removed characters smoothly
5. **Scales the container** to accommodate the new text dimensions

## Customization

You can customize the animation behavior with props:

<CodeGroup>
```tsx React
<TextMorph
  duration={600}
  ease="cubic-bezier(0.19, 1, 0.22, 1)"
  scale={true}
  as="h1"
  className="my-text"
>
  {text}
</TextMorph>
```

```vue Vue
<TextMorph
  :text="text"
  :duration="600"
  ease="cubic-bezier(0.19, 1, 0.22, 1)"
  :scale="true"
  as="h1"
  class="my-text"
/>
```

```svelte Svelte
<TextMorph
  {text}
  duration={600}
  ease="cubic-bezier(0.19, 1, 0.22, 1)"
  scale={true}
  as="h1"
  class="my-text"
/>
```
</CodeGroup>
