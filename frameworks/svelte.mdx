## TextMorph component

The `TextMorph` component is built with native Svelte syntax and reactive statements.

### Basic usage

```svelte
<script>
  import { TextMorph } from 'torph/svelte';
  
  let text = 'Hello World';
</script>

<TextMorph {text} class="text-4xl font-bold" />
```

### Props

<ParamField path="text" type="string" required>
  The text content to display and animate.
</ParamField>

<ParamField path="class" type="string" default="">
  CSS class name to apply to the container element.
</ParamField>

<ParamField path="style" type="string" default="">
  Inline styles string to apply to the container element.
</ParamField>

<ParamField path="as" type="string" default="div">
  The HTML element to render as the container.
</ParamField>

<ParamField path="locale" type="Intl.LocalesArgument" default="en">
  Locale for text segmentation. Used by `Intl.Segmenter` to properly split text into graphemes or words.
</ParamField>

<ParamField path="duration" type="number" default={400}>
  Animation duration in milliseconds.
</ParamField>

<ParamField path="ease" type="string" default="cubic-bezier(0.19, 1, 0.22, 1)">
  CSS easing function for the animation.
</ParamField>

<ParamField path="scale" type="boolean" default={true}>
  Whether to apply a subtle scale effect during animation.
</ParamField>

<ParamField path="debug" type="boolean" default={false}>
  Enable debug mode to visualize element boundaries with colored outlines.
</ParamField>

<ParamField path="disabled" type="boolean" default={false}>
  Disable animations and render text updates instantly.
</ParamField>

<ParamField path="respectReducedMotion" type="boolean" default={true}>
  Respect the user's `prefers-reduced-motion` setting and disable animations when enabled.
</ParamField>

<ParamField path="onAnimationStart" type="() => void">
  Callback function called when an animation starts.
</ParamField>

<ParamField path="onAnimationComplete" type="() => void">
  Callback function called when an animation completes.
</ParamField>

## Reactive updates

The component uses Svelte's reactive statement syntax (`$:`) to automatically trigger animations when the text changes.

```svelte
<script>
  import { TextMorph } from 'torph/svelte';
  
  let count = 0;
  $: text = `Count: ${count}`;

  setInterval(() => {
    count++;
  }, 1000);
</script>

<TextMorph {text} />
```

<Note>
  The component internally uses `$: if (morphInstance) morphInstance.update(text)` to watch for text changes and trigger animations automatically.
</Note>

## Lifecycle

The component handles initialization and cleanup using Svelte's lifecycle functions:

- **onMount**: Creates the TextMorph instance and performs the initial render
- **onDestroy**: Destroys the instance and cleans up animations

```svelte
<script>
  import { onMount, onDestroy } from 'svelte';
  import { TextMorph } from 'torph/svelte';
  
  // The component handles these internally:
  // onMount(() => {
  //   // TextMorph instance is created
  //   // Initial text is rendered
  // });

  // onDestroy(() => {
  //   // TextMorph instance is destroyed
  //   // Animations are cleaned up
  // });
</script>
```

## TypeScript

Torph provides full TypeScript support for Svelte applications.

### Type definitions

```svelte
<script lang="ts">
  import { TextMorph } from 'torph/svelte';
  import type { TextMorphProps } from 'torph/svelte';
  
  let text: string = 'Hello';
</script>
```

### Interface

```ts
interface TextMorphProps extends Omit<TextMorphOptions, 'element'> {
  text: string;
  class?: string;
  style?: string;
  as?: string;
}
```

## SvelteKit

The Svelte component works seamlessly with SvelteKit.

```svelte
<script>
  let text = 'Welcome to SvelteKit';
</script>

<TextMorph {text} class="hero-text" />
```

<Tip>
  The component only runs animations in the browser. During SSR, it renders the initial text without animation, ensuring a smooth user experience.
</Tip>

## Prop shorthand

You can use Svelte's prop shorthand syntax when the variable name matches the prop name:

```svelte
<script>
  let text = 'Hello';
  let duration = 600;
  let debug = true;
</script>

<!-- Shorthand syntax -->
<TextMorph {text} {duration} {debug} />

<!-- Equivalent to: -->
<TextMorph text={text} duration={duration} debug={debug} />
```
