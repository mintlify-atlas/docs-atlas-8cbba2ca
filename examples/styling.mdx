## Using className

Style your text morph component with CSS classes.

<Tabs>
  <Tab title="React">
    ```tsx
    import { TextMorph } from 'torph/react';
    import './App.css';

    export default function App() {
      return (
        <TextMorph className="hero-text">
          Styled Text Morph
        </TextMorph>
      );
    }
    ```

    ```css
    /* App.css */
    .hero-text {
      font-size: 3rem;
      font-weight: bold;
      color: #6366f1;
      text-align: center;
    }
    ```
  </Tab>
  <Tab title="Vue">
    ```vue
    <template>
      <TextMorph text="Styled Text Morph" class="hero-text" />
    </template>

    <script setup lang="ts">
    import { TextMorph } from 'torph/vue';
    </script>

    <style scoped>
    .hero-text {
      font-size: 3rem;
      font-weight: bold;
      color: #6366f1;
      text-align: center;
    }
    </style>
    ```
  </Tab>
  <Tab title="Svelte">
    ```svelte
    <script lang="ts">
      import { TextMorph } from 'torph/svelte';
    </script>

    <TextMorph text="Styled Text Morph" class="hero-text" />

    <style>
      :global(.hero-text) {
        font-size: 3rem;
        font-weight: bold;
        color: #6366f1;
        text-align: center;
      }
    </style>
    ```
  </Tab>
</Tabs>

## Inline styles

Apply inline styles directly to the component.

<Tabs>
  <Tab title="React">
    ```tsx
    import { TextMorph } from 'torph/react';

    export default function App() {
      return (
        <TextMorph
          style={{
            fontSize: '2.5rem',
            fontWeight: 'bold',
            color: '#ec4899',
            letterSpacing: '0.05em',
          }}
        >
          Inline Styled Text
        </TextMorph>
      );
    }
    ```
  </Tab>
  <Tab title="Vue">
    ```vue
    <template>
      <TextMorph 
        text="Inline Styled Text" 
        :style="{
          fontSize: '2.5rem',
          fontWeight: 'bold',
          color: '#ec4899',
          letterSpacing: '0.05em',
        }"
      />
    </template>

    <script setup lang="ts">
    import { TextMorph } from 'torph/vue';
    </script>
    ```
  </Tab>
  <Tab title="Svelte">
    ```svelte
    <script lang="ts">
      import { TextMorph } from 'torph/svelte';
    </script>

    <TextMorph 
      text="Inline Styled Text" 
      style="font-size: 2.5rem; font-weight: bold; color: #ec4899; letter-spacing: 0.05em;"
    />
    ```
  </Tab>
</Tabs>

<Note>
  In Svelte, the `style` prop accepts a string. In React and Vue, it accepts an object.
</Note>

## Gradient text

<Tabs>
  <Tab title="React">
    ```tsx

    import './App.css';

    export default function App() {
      return (
        <TextMorph className="gradient-text">
    ```

    ```css
      -webkit-text-fill-color: transparent;
    }
    ```

## Responsive sizing

Make text responsive using CSS or framework utilities.

<Tabs>
  <Tab title="React">
    ```tsx
    import { TextMorph } from 'torph/react';
    import './App.css';

    export default function App() {
      return (
        <TextMorph className="responsive-text">
          Responsive Design
        </TextMorph>
      );
    }
    ```

    ```css
    /* App.css */
    .responsive-text {
      font-size: 1.5rem;
      font-weight: bold;
    }

    @media (min-width: 768px) {
      .responsive-text {
        font-size: 2.5rem;
      }
    }

    @media (min-width: 1024px) {
      .responsive-text {
        font-size: 3.5rem;
      }
    }
    ```
  </Tab>
  <Tab title="Vue">

## Debug mode

Enable debug mode to visualize the text morph boundaries and individual character elements.

<Tabs>
  <Tab title="React">
    ```tsx
    import { TextMorph } from 'torph/react';

    export default function App() {
      return (
        <TextMorph debug>
          Debug Mode Active
        </TextMorph>
      );
    }
    ```
  </Tab>
  <Tab title="Vue">
    ```vue
    <template>
      <TextMorph text="Debug Mode Active" :debug="true" />
    </template>

    <script setup lang="ts">
    import { TextMorph } from 'torph/vue';
    </script>
    ```
  </Tab>
  <Tab title="Svelte">
    ```svelte
    <script lang="ts">
      import { TextMorph } from 'torph/svelte';
    </script>

    <TextMorph text="Debug Mode Active" debug={true} />
    ```
  </Tab>
</Tabs>

<Note>
  Debug mode adds magenta outlines to the root container and cyan outlines to individual text items. This helps visualize how Torph segments and animates your text.
</Note>
